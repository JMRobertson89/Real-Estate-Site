# Real-Estate-Site


## Description
    Full-Stack website built with Django. Users can register, login, view listings, and make inquiries. Admins can add/edit listings, add/edit employees, and see view inquiries.

## Technologies 

- Python
- Django
- JavaScript
- SQL
- PostgreSQL
